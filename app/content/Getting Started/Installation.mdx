# Installation

This guide covers installing and setting up FairCredit for development.

## Prerequisites

Before installing FairCredit, make sure you have:

- Node.js 18+ installed
- npm, pnpm, or yarn package manager
- Git installed
- A Solana wallet for testing

## Installation Steps

### 1. Clone the Repository

```bash
git clone [repository-url]
cd FairCredit/app
```

### 2. Install Dependencies

Using npm:
```bash
npm install
```

Using pnpm:
```bash
pnpm install
```

Using yarn:
```bash
yarn install
```

### 3. Install Nextra Dependencies

If Nextra packages aren't installed, run:

```bash
npm install nextra nextra-theme-docs
npm install --save-dev pagefind
```

### 4. Run Development Server

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to see the application.

## Documentation

The documentation is available at [http://localhost:3000/docs](http://localhost:3000/docs).
