# Getting Started with FairCredit

Welcome to FairCredit! This guide will help you get started with using FairCredit for credential management.

## What is FairCredit?

FairCredit is a decentralized credential management system built on Solana. It allows you to:

- Create verifiable credentials on-chain
- Manage credential providers through hubs
- Verify credentials instantly
- Endorse credentials as a supervisor

## Prerequisites

Before getting started, make sure you have:

- A Solana wallet (Phantom, Solflare, etc.)
- Some SOL for transaction fees
- Basic understanding of blockchain concepts
